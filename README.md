# SAST-evento 管理端

基于 React 的**企业级活动**管理后台系统，使用 Vite 构建工具和 React Router 实现路由管理。

## 技术栈

- ⚛️ React 19
- 🚀 Vite 6
- 🧭 React Router v7
- 🦺 TypeScript (可选)
- 🛠 ESLint + Prettier
- 🐶 Husky + Commitlint

## 快速开始

### 环境要求

- Node.js >= 20.x
- PNPM >= 8.x

### 安装依赖

```bash
pnpm install
```

### 提交项目

```bash
pnpm commit

```
